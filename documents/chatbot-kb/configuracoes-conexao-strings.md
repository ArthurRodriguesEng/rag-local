# Conexão das Strings

No sistema, todas as **entradas dos inversores** são lidas, porém apenas as **entradas efetivamente em uso** são exibidas, sendo mapeadas individualmente para cada **string**.

Caso ocorra qualquer alteração na ligação das strings em campo, acesse a opção **Conexão das Strings**, selecione a **UFV** e o **inversor** correspondente, informe a **nova posição de conexão** da string e clique em **confirmar** para atualizar o mapeamento no sistema.

Essa configuração garante que as informações exibidas nas telas de análise reflitam corretamente a topologia real do sistema fotovoltaico.

![Conexão das Strings](/assets/articles/configuracoes/conexao-strings/conexao-strings1.gif)

> **Importante:** Ao preencher duas strings com a mesma posição de conexão, haverá duplicidade de dados no sistema.
