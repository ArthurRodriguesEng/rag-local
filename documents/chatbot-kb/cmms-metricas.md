# Métricas

<script setup>
import { withBase } from 'vitepress'
</script>

Na tela **Métricas**, é possível visualizar algumas métricas para analisar o desempenho da equipe. Da mesma forma que na tela de ordens de serviço, é possível selecionar os seguintes filtros:

- **Tipo de Manutenção**
- **Atribuição** (Criadas pelo usuário ou atribuídas a ele)
- **Status** (Criado, Aberto, Em andamento, Finalizado)
- **Estado**
- **Usina**
- **O&M**
- **Usuários Atribuídos**
- **Criado por**
- **Categoria e Subcategoria**
- **Sistema**
- **Categoria**
- **Subcategoria**
- **ID**
