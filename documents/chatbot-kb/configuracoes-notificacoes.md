# Notificações

## Grupos de Notificação

Para configurar o envio de notificações de alarmes, é necessário criar um **Grupo de Notificação**. Nesse processo, o usuário define os **filtros de alarme** que irão acionar as notificações, como a **UFV**, o **tipo de equipamento** e o **nível de severidade**.

Os grupos de notificação permitem segmentar os alarmes de acordo com critérios operacionais, garantindo que cada evento relevante seja encaminhado às pessoas ou canais adequados.

![Notificações](/assets/articles/configuracoes/notificacoes/notificacoes1.gif)

## Canais de Alarme

O sistema disponibiliza dois canais para o recebimento de alarmes: **e-mail** e **Telegram**.

Para configurar o recebimento de notificações via **Telegram**, clique em **vincular conta do Telegram** e realize o scan do **QR Code** exibido na tela. Após a vinculação, os alarmes serão enviados automaticamente para a conta associada.

Para receber notificações via **e-mail**, basta marcar a opção correspondente no grupo de notificação. Os alarmes serão enviados para o endereço de e-mail cadastrado no sistema.
